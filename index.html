Software Engineering Git Practice

Welcome to the Git practical session! This interactive tutorial will guide you through the essentials of Git and GitHub workflows.

Your Task

  1. Fork this repository:
    • Visit https://github.com/alfhisa/se-tutrial-git and click "Fork".
  2. Clone the forked repository:
    • git clone https://github.com/alfhisa/se-tutrial-git.git
    • cd se-tutrial-git
  3. Create a new branch:
    • git checkout -b latihan-git (name the branch as you prefer)
  4. Edit this file (index.html) and add your name and insight:
    • Be as creative as possible with your edits.
    • Feel free to add additional files if desired.
  5. Commit and push your changes:
    • git add .
    • git commit -m "Edit the index.html file" (customize the message)
    • git push origin latihan-git (use your branch name)
  6. Open a pull request on GitHub:
    • Visit your forked repository.
    • Click "Compare & pull request".
    • Add a title and description.
    • Submit the pull request.
