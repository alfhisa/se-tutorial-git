Welcome to Git Mastery

This interactive tutorial will guide you through the essential Git and GitHub workflows that every software engineer should know. Follow the steps below to complete your hands-on practice session.

Your Task

  1. Fork the repository

    Visit https://github.com/alfhisa/se-tutrial-git and click the "Fork" button in the top-right corner.

  2. Clone the forked repository
    git clone https://github.com/alfhisa/se-tutrial-git.git
    cd se-tutrial-git
  3. Create a new branch
    git checkout -b latihan-git

    You may name the branch as you like

  4. Edit this file and add your name and insight

    Edit the file as creatively as possible

    You can also add more files if you want

  5. Commit and Push your changes
    git add .
    git commit -m "Edit the index.html file"

    You may change the commit message

    git push origin latihan-git

    Adjust with your branch name

  6. Open a pull request on GitHub

    Visit your forked repo

    Click "Compare & pull request"

    Fill out the title and description

    Submit the pull request
